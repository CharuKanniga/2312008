AffordMed Campus Evaluation – Full Stack


This project was developed as part of the AffordMed Campus Evaluation. It includes a reusable logging middleware, a backend service, and a frontend notification application. The project demonstrates API integration, database connectivity, and a responsive user interface.

Tech Stack

- React
- Vite
- Node.js
- Express.js
- SQLite
- Sequelize

Features

- Reusable logging middleware
- REST API backend
- Notification dashboard
- Category-based filtering
- Backend and frontend integration
- Error handling and validation

Run the Project

Backend

cd notification-app-be
npm install
npm start

Frontend

cd notification-app-fe
npm install
npm run dev

Logging Middleware

cd logging-middleware
npm install
node test.js
